# dotfiles

A polished Linux dotfiles collection for Hyprland, Kitty, Rofi, and Neovim.

## What is included

- `hypr/` — Hyprland compositor configuration and helper scripts
- `kitty/` — Kitty terminal configuration with theme variants
- `rofi/` — Rofi launcher theme and custom power menu
- `nvim/` — Neovim setup powered by LazyVim
- `opencode/` — OpenCode/OpenAgent AI tooling configuration

## Key components

### hypr/
- `hyprland.conf` — main Hyprland settings, startup apps, environment vars, and window styling
- `hyprlock.conf` — lock screen settings
- `hyprpaper.conf` — wallpaper/background settings
- `hypridle.conf` — idle behavior and timeout settings
- `scripts/screenrecord-toggle.sh` — quick start/stop screen recording with `wf-recorder`

### kitty/
- `kitty.conf` — terminal configuration with JetBrains Mono, blur, transparency, and minimal decoration
- theme files — adaptive themes for dark, light, and no-preference modes

### rofi/
- `config.rasi` — main launcher theme using a rounded template
- `colors/` — color palettes for different Rofi themes
- `powermenu/` — a custom power menu script with shutdown, reboot, lock, suspend, and logout actions

### nvim/
- `init.lua` — bootstraps LazyVim and local configuration
- `lazyvim.json` / `lazy-lock.json` — plugin dependency management
- `lua/config/` — curated Neovim settings, keymaps, plugins, and editor behavior
- `lsp.lua` — language server setup

### opencode/
- `opencode.json` — OpenCode configuration targeting `minimax` and fallback AI models
- `oh-my-openagent.json` — OpenAgent integration settings
- `skills/` — custom OpenCode skill definitions

## Usage

1. Link or copy folders to the appropriate config locations:
   - `~/.config/hypr/`
   - `~/.config/kitty/`
   - `~/.config/rofi/`
   - `~/.config/nvim/`
2. Install required packages: Hyprland, Kitty, Rofi, Neovim, and any related tools.
3. Reload or restart your session to apply the Hyprland and Rofi settings.
4. Open Neovim to confirm plugin and editor configuration.

## Notes

- `hypr/hyprland.conf` is based on an autogenerated template and can be customized fully.
- The Rofi setup uses a rounded theme and shared styling fragments for consistency.
- Kitty includes adaptive theme files for different display modes and aesthetic options.
- Neovim uses LazyVim as its core framework with local customizations loaded from `lua/`.
